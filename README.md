# 3.1_ExtendedReality_Final

A-Frame VR project with Vite build tool.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Technologies

- **Vite** - Fast build tool and dev server
- **A-Frame** - Web framework for building VR experiences (v1.7.0)

## Project Structure

- `index.html` - Main HTML file with A-Frame scene
- `main.js` - Entry point that imports A-Frame and custom components
- `vite.config.js` - Vite configuration
- Custom components: `button.js`, `color-change.js`, `event-manager.js`, `menu.js`, `pinchable.js`, `pressable.js`, `size-change.js`, `slider.js`
